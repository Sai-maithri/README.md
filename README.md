# HTML Projects Collection

A collection of interactive web projects built with HTML, CSS, and JavaScript.

##  Project Structure

```
├── bio.html / bio.css          # Biology subject project
├── matrix.html / matrix.css / matrix.js  # Matrix operations calculator
├── subject1.html / subject1.css # Subject 1
├── subject2.html / subject2.css # Subject 2
├── subject3.html / subject3.css # Subject 3
├── subject4.html / subject4.css # Subject 4
├── subject5.html / subject5.css # Subject 5
```

##  Features

- **Matrix Operations** — Perform addition, subtraction, and multiplication on 2×2 matrices
- **Interactive UI** — User-friendly forms with instant results
- **Responsive Design** — Works on various screen sizes

##  Technologies

- HTML5
- CSS3
- JavaScript

## Usage

Open any `.html` file in a web browser to view the project.
